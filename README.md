
# gnome-theme-macos

## 部署方法

```shell
https://github.com/tcbaby/gnome-theme-macos.git
cd gnome-theme-macos
sh install.sh
```

## 设置方法

部署完成后，按下ALT+F2输入r重新加载桌面，然后打开Tweaks-tool进行设置

![](/images/1.png)
![](/images/2.png)

打开Dash to Dock设置Dash位置、图标大小等
